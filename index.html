PagamentoAgil.Net

Este projeto de classe permite a criação de boleto bancário através da integração com a plataforma de cobrança Pagamento Ágil (www.pagamentoagil.com.br).

Artigo e Ajuda

Dê uma olha no site https://ivaldojunior.wordpress.com/2015/07/17/net-saiba-como-gerar-um-boleto-bancario-com-apenas-duas-linhas-de-codigo/

Observações

Antes de utilizar esta biblioteca, se faz necessário a assinatura do serviço do site www.pagamentoagil.com.br.

Pré-requisitos

.Net framework 4.5 (ou superior)
